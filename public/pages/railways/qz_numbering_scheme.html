Railways of Qevelia numbering scheme

Locomotives

The railways of Qevelia have a numbering scheme for their trains that looks like this: CCCC NNNNK

The class number consists of a four digit number:

First Digit

The first digit indicates the type of locomotive.

  • 0: Steam locomotive tenders
  • 1: Steam locomotives
  • 2: Electric locomotives
  • 3: Non-electric locomotives
  • 4: Electric Railcars
  • 5: Non-electric railcars
  • 6: Intermediate cars of MUs
  • 7: (Passenger Rollingstock)
  • 8: (Freight rollingstock)
  • 9: Non-revenue equipment

The second digit indicates some broad technical aspects.

Fifth–Eighth Digits

These digits are the locomotive's serial number, usually starting with 0001. The thousand's digit (or fifth digit) may be used to represent other information; experimental variants of locomotives are often numbered in the 5xxx series. For multiple unit intermediate cars (6xxx), the fifth digit is identical to the second digit for the whole set classification, so a typical EMU set may have numbers such as:
4250 00118 = 6150 20210 + 6550 20215 + 6550 20227 + 6150 20222.

Ninth Digit

The final digit, K, is a check digit used to record the locomotive's id in computer entries. The check digit is calculated by multiplying each digit by its position from right to left, adding all but the rightmost together. The result is then divided by the length of the number, and the remainder becomes the rightmost, or check, digit.

Passenger Cars

Passenger cars have a similar numbering system that is designed to blend in with the locomotive one. The scheme is as follows:

First Digit

The first digit is always 7.

Second Digit

The second digit indicates the wheel arrangement.

  • 1: Fixed wheelbase
  • 2: Two 2-axle bogies
  • 3: Two 3-axle bogies
  • 4: Articulated, single wheels
  • 5: Articulated, 2-axle bogies
  • 6: Articulated, 3-axle bogies
  • 7: Articulated, 2-axle bogies at ends, 3-axle intermediate bogies

Third and Fourth Digits

The third and fourth digits indicate information about the type of car:

  • 01–09: Private car
  • 10–19: Premium class intercity car
  • 20–29: Standard class intercity car
  • 30–39: Premium class sleeping car
  • 40–49: Standard class sleeping car
  • 50–59: Premium class with luggage and/or guard's compartment (includes driving trailers)
  • 60–69: Standard class with luggage and/or guard's compartment (includes driving trailers)
  • 70–74: Buffet car
  • 75–79: Dining car
  • 80–84: Mail sorting van
  • 85–89: Luggage van
  • 90–93: Power van
  • 94: Heating van
  • 95–98: Specialised/miscellaneous passenger cars
  • 99: Crew car

Fifth–Ninth digits

These are the serial number and check digit of the individual item of rollingstock, identical in form and function to the locomotive serial number, described above.

Freight Rollingstock

QJ freight rollingstock has a similar type of numbering code to the locomotives and passenger rollingstock, with the exception that the code has up to four digits to indicate class. The first digit is always 8, the second describes the type, the third gives further information, and the fourth is to differentiate individual series with otherwise identical characteristics.

Second Digit

  • 0: Enclosed wagons, standard types
  • 1: Enclosed wagons, non-standard types
  • 2: Open wagons, standard types
  • 3: Open wagons, hoppers
  • 4: Refrigerated wagons
  • 5: Tank wagons
  • 6: Flat wagons
  • 7: Container wagons
  • 8: Miscellaneous wagons
  • 9: Non-revenue equipment

Fourth Digit

The fourth digit represents individual types within above classifications.

Fifth–Ninth digits

These are the serial number and check digit of the individual item of rollingstock, identical in form and function to the locomotive serial number, described above.

Examples

2242 00238
Electric locomotive, Dual voltage (800V= + 16kV 20Hz), Single unit modern series, 23rd in class, check digit is 8.
2×9 + 2×8 + 4×7 + 2×6 + 0×5 + 0×4 + 2×3 + 3×2 = 80
80 mod 9 = 8
Therefore check digit is 8.
7254 00615
Passenger car with two 2-axle bogies, premium class with luggage and/or guard's compartment, 61st in class, check digit is 5.
7×9 + 2×8 + 5×7 + 4×6 + 0×5 + 0×4 + 6×3 + 1×2 = 140
140 mod 9 = 5
Therefore check digit is 5.

8×9 + 2×8 + 1×7 + 3×6 + 1×5 + 5×4 + 7×3 + 6×2 = 150

Therefore check digit is 6.
