Qevelian Railway Signalling

Cab Signalling

There are two cab signalling systems in use on the Qevelian railways. The older system uses beacons powered by magnetic induction that transmit the maximum speed to the train. This has been augmented with a newer system that can transmit more detailed information from cable loops mounted between the rails.

Each of these systems are described in greater detail on their own page:

Speed Control by Beacon
The older system that uses magnetic beacons to transmit signal information to the trains.
Linear Train Control
A newer system, developed for high-speed trains, that continuously transmits speed information to the trains.
