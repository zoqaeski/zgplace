About this site

This site was written by Zoqaeski, and it was mostly hand coded using Vim. The site was optimised for modern browsers such as Chromium and Firefox, and has been tested with the latest (relatively) stable versions of each of these. The pages are rendered by a PHP application that I wrote, which in turn relies on an excellent DOM parser.

Both the CSS and XHTML have been validated with the W3C's validator service to ensure compliance. Please note that some features may not work in other browsers to those listed above.
